**Agrosmart AI changes the narrative.**

---

## 🔥 What It Does
An all-in-one AI companion engineered to detect plant diseases instantly and prescribe immediate, actionable remedies.

* 📸 **Instant Visual Diagnosis:** Upload any picture of a distressed leaf, and watch the AI analyze the anomaly in seconds.
* 🤖 **Next-Gen Neural Logic:** Backed by the cutting-edge Google Gemini Vision API to guarantee high-accuracy disease identification.
* 💊 **Prescriptive Remedies:** It doesn’t just name the disease; it tells you exactly how to fix it, organic treatment protocols, and chemical safeguards.
* 🌐 **Global Accessibility:** Fully deployed in the cloud. Optimized to run flawlessly on computers, laptops, and mobile smartphones out in the open field.

---

## 🛠️ The Engine Under the Hood
Built using a robust, clean, and modern Python ecosystem:

* **Frontend Architecture:** `Streamlit` (For a sleek, reactive, user-centric dashboard)
* **AI Cognitive Core:** `Google Gemini API` (Advanced computer vision multimodal processing)
* **Image Processing:** `Pillow (PIL)` (For lightning-fast image handling and array processing)
* **Environment Security:** `Python-Dotenv` (Keeping sensitive production API keys locked down and secure)

## 🚀 Experience It Locally

Want to fire up this engine on your local machine? Run the following sequence in your command terminal:

1. **Clone the repository:**
   ```bash
   git clone https://github.com.git
   cd AGROSMART-AI
   ```

2. **Activate your virtual environment and install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Inject your Credentials:**
   Create a `.env` file in the root folder and drop your Gemini key:
   ```text
   GOOGLE_API_KEY = "your_secret_gemini_api_key"
   ```

4. **Ignite the Application:**
   ```bash
   streamlit run app_streamlit.py
   ```
